  📝 CẤU HÌNH QUAN TRỌNG (.env)
═══════════════════════════════════════════════════════════════════════════════════

  PORT=3000                              → Cổng server
  BETTING_UNIT=1000                      → Đơn vị cược (VND)
  PREDICTION_POLL_MS=1500                → Kiểm tra dự đoán (ms)
  BET_ADVICE_MIN_CONFIDENCE=57           → Min confidence
  BET_ADVICE_MAX_BANKROLL_PERCENT=8      → Max cược (%)
  LOG_LEVEL=info                         → Mức log (error/warn/info/debug)
  LOG_REQUESTS=false                     → Bật log chi tiết

  💡 TIPS:
     • Tăng BETTING_UNIT → Cược lớn hơn
     • Giảm PREDICTION_POLL_MS → Nhanh hơn (nhưng dùng CPU nhiều)
     • Tăng BET_ADVICE_MIN_CONFIDENCE → Gợi ý ít nhưng chính xác hơn

  🐛 TROUBLESHOOTING - GIẢI QUYẾT VẤN ĐỀ
═══════════════════════════════════════════════════════════════════════════════════

  ❌ "Node.js is not recognized"
     → Cài Node.js: https://nodejs.org/
     → Khởi động lại PowerShell

  ❌ "npm ERR! code ENOENT"
     → Chắc chắn đang ở đúng thư mục: D:\Users\Asus\Downloads\tool\tx-prediction-backend
     → Kiểm tra có file package.json

  ❌ "Address already in use :::3000"
     → Port 3000 đang dùng
     → Cách 1: Đổi PORT=3001 trong .env
     → Cách 2: Get-Process node | Stop-Process

  ❌ "Cannot find module 'express'"
     → npm install chưa hoàn tất
     → Chạy: npm install

  ❌ Port 3000 không mở được
     → Kiểm tra server có chạy: Get-Process node
     → Nếu không → npm run start

  👉 XEM CHI TIẾT: GUIDE.md - Phần Troubleshooting
